## Method

The `stats` method provides statistical information about the index, returning the following fields:

- `vector_count`: The total number of vectors in the index.
- `pending_vector_count`: The number of vectors that are currently pending (not yet fully processed).
- `index_size`: The size of the index in bytes.

## Stats Example

```python
from upstash_vector import Index

index = Index.from_env()

# Get statistical information about the index
stats_result = index.stats()

# Display the stats result
print("Vector Count:", stats_result.vector_count)
print("Pending Vector Count:", stats_result.pending_vector_count)
print("Index Size:", stats_result.index_size)
```
